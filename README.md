# Timer App

シンプルなカウントダウンタイマーのWebアプリ。

## 技術スタック

- React 18
- Vite
- Tailwind CSS

## 機能

- カウントダウンタイマー（分・秒で設定）
- プリセットボタンで時間を加算（3h〜10s）
- プログレスリング表示
- スタート / 一時停止 / リセット

## セットアップ

```bash
npm install
npm run dev
```

ブラウザで `http://localhost:5173` を開く。

## コマンド

| コマンド | 説明 |
|---|---|
| `npm run dev` | 開発サーバー起動 |
| `npm run build` | プロダクションビルド |
| `npm run preview` | ビルド結果のプレビュー |
